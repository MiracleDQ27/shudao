# 设计规范

> 视觉风格、配色、布局规范

## 整体风格

像素 RPG 角色面板 + HD-2D 光影混合感（参考《八方旅人》）

- 深色背景 + 微妙粒子飘动
- 像素块感边框和分割线
- 暖色为主色调
- 雷达图等数据可视化为平滑现代风格（对比产生亮点）

## 字体

- **英文/数字**：`Press Start 2P`（Google Fonts）
- **中文**：`Noto Serif SC`（思源宋体）作为降级
- 字号：主标题 24px、正文 14px、标签 10px

## 布局

主面板从上到下：
1. 角色信息（头像、等级、XP条、修习天数）
2. 七维雷达图（Canvas，平滑动画）
3. Boss地图（当前副本）
4. 书域地图（7 区域网格）
5. Boss战记录（最近 5 条）
6. 书架 + 修行路线
7. 修习日历（底部）

## 交互规范

- Hover 效果：轻微上浮 + 边框发光
- 点击展开：书架、Boss 历史可折叠
- 雷达图载入动画：从 0 到目标值的平滑过渡（1.5s）
- 粒子效果：背景自动飘动，不干扰内容阅读
