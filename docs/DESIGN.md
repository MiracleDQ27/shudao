# 书道 · 读书修行系统设计

## 一、核心循环

```
读书（微信读书）→ 聊理解（IMA DeepSeek）→ IMA 自动追加笔记 →
Claude 定期结算（拉微信读书API + IMA笔记）→ 写入 data.json → 网页可视化
```

三端分工：
| 端 | 职责 |
|----|------|
| 手机端（微信读书 + IMA DeepSeek） | 读 + 聊 + 自动记录 |
| Claude | 拉数据 → 交叉验证 → 计算 XP/维度/Boss → 写 data.json |
| 网页 | 纯可视化 data.json |

### 新书分类
每本书入库时定三个属性：
| 属性 | 说明 |
|------|------|
| 主维 | 核心学科（必填） |
| 副维 | 次要视角（可选，最多1个） |
| 难度 | 1-5 档 |

**难度分档：**
| 档位 | 类型 | XP 倍率 |
|------|------|---------|
| 1 | 通俗读物、随笔 | ×0.7 |
| 2 | 一般社科、通识 | ×1.0 |
| 3 | 严肃学术、系统著作 | ×1.3 |
| 4 | 经典原著、艰深理论 | ×1.6 |
| 5 | 大部头、跨学科巨著 | ×2.0 |

> 谁判定：Claude 根据书名和内容判断主维/副维/难度，写入 data.json。用户可纠正。

```
整本书读完额外奖励：+30（主维）
```

## 四、Boss 链机制

三层反馈：
| 类型 | 触发时机 | 定位 |
|------|----------|------|
| 小怪 | 每次读书写完理解 | 即时反馈 |
| 章 Boss | 一章读完 | 章节里程碑 |
| 书 Boss | 整本书读完 | 全书终极结算 |

### 战胜条件（Claude 判定）

| 评级 | 标准 | 结果 |
|------|------|------|
| 优秀 | 理解准确、有独立思考 | 完美击杀 |
| 合格 | 方向正确、无明显偏差 | 普通击杀 |
| 偏差 | 存在误解或遗漏 | 需重新理解 |
| 跳过 | 未写理解直接读完 | 无奖励 |

### Boss 奖励（Claude 结算时计算）
```
小怪 XP    = 难度倍率 × 20 × 评分系数
章 Boss XP = 难度倍率 × 30
书 Boss XP = 难度倍率 × 50

评分系数：
  优秀 = 1.5
  合格 = 1.0
```

## 五、复习触发机制

### 流程
```
每次读书聊完 → IMA DeepSeek 判断：
  有没有自然关联的旧知识点？
    ↓
  有 → 出一道简答题考用户
    ↓
  答对 → +10 XP，该知识点巩固+1
  答错 → "需要回去翻翻"，无 XP，巩固清零
  连续 3 次答对 → 标记「铭刻」，不再考
```

### 出题频率
不固定，约每 3-4 次读书触发 1 次。有自然关联才出，不强行。

### 知识点追踪（data.json knowledgePoints 字段）
```json
{
  "knowledgePoints": [
    {
      "id": "kp_001",
      "content": "汉代三省六部制中相权一分为三的制衡逻辑",
      "book": "中国历代政治得失",
      "dimension": "政道",
      "correctCount": 2,
      "status": "active"
    }
  ]
}
```
- correctCount 答错即清零
- status 变为「铭刻」后不再出题

## 八、Claude 结算流程

```
你喊"同步书道"
    ↓
① 拉微信读书 API → 阅读进度、时长、修习天数
② 拉 IMA 笔记 → 理解评分、书域、复习记录
③ 交叉验证 → 进度 ↔ 阅读范围
④ 计算：
   ├── XP = 基础XP × 难度倍率 × 修习加成 + 复习XP
   ├── 维度 = 理解评分 × 3
   ├── Boss = 按进度触发（小怪/章Boss/书Boss）
   └── 等级 = 累计XP查表
⑤ 写入 data.json
⑥ 汇报摘要
```

## 九、IMA DeepSeek 记录指令（复制喂给 IMA）

```
你是「书道」读书修行系统的 AI 记录官。

每次和我聊完读书内容后，在我的「书道」笔记本中，
找到对应书名的笔记追加更新。没有则新建。

追加格式（每次追加一节）：
---
· 阅读范围：{书名}的{章节或页码、内容范围} · YYYY-MM-DD
· 所属书域：文道/史道/哲道/政道/世道/心道/器道（可注主/副，如"史道（主）· 政道（副）"）
· 聊了什么：{用 2-3 句话概括讨论的核心内容}
· 理解评分：{你的判定，1-10}
    8-10：理解深入，有独立思考，能串联其他知识
    5-7：方向正确，基本吃透
    3-4：有一些误解或重要遗漏
    1-2：几乎没理解，需要重读
· 复习触发：是/否（若有，注明关联哪个旧知识点及结果）
· 评价：{一句话，指出亮点或问题}

我的分享是零散的、跳跃的，从整体判断。
```
